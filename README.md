# 🚀 Prnv Rooms

A modern **Real-Time Communication & Collaboration Platform** built with **React, Node.js, Express, MongoDB, Socket.IO, and WebRTC**. Prnv Rooms enables users to communicate seamlessly through secure authentication, real-time messaging, and online meeting rooms.

🌐 **Live Demo:** https://prnvrooms.vercel.app/

---

## 📌 Overview

Prnv Rooms is designed to provide a smooth and interactive communication experience. The platform combines modern web technologies with real-time capabilities, allowing users to connect instantly and collaborate efficiently.

Whether it's chatting with teammates, joining meeting rooms, or interacting in real time, Prnv Rooms offers a scalable and responsive solution.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/prnvrdyy-io/PrnvRooms.git

cd PrnvRooms
```

---

## Install Dependencies

### Client

```bash
cd client

npm install
```

### Server

```bash
cd ../server

npm install
```

---

# 🔑 Environment Variables

## Client (.env)

```env
VITE_API_URL=your_backend_url
```

---

## Server (.env)

```env
PORT=5000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:5173
```

*(Add any additional environment variables your project requires.)*
